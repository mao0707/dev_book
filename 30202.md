变量的命名规则和类方法的命名规则非常接近，使用全小写，使用下划线分割， 并且应该明确指出该变量的用途。非常短的无意义的变量只应该在 for 循环中作为迭代器使用。

错误的:

```PHP

$j = 'foo';         // 单个变量只用在for()等循环中

$Str                // 包含大写字母

$bufferedText       // 使用语义，可用缩短的语义

$groupid            // 使用驼峰

$name_of_last_city_used // 太长

```

正确的:

```PHP

for ($j = 0; $j < 10; $j++)

$sStr

$sBuffer

$iGroupId

$sLastCity

```

规则：

```PHP

使用驼峰命名规则

首字母使用类型缩写

o(object)    // 类

a(array)     // 数组

b(boolen)    //布尔

s(string)    // 字符串

i(int)       // 整形

f(float)     //符点

r(resource)  //资源

```
