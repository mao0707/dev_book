* 部门同步请求地址
```
http://oa.eol.com.cn/api.php?mod=topdepartment
```
* 用户同步请求地址
```
http://oa.eol.com.cn/api.php?mod=topadmin
```

使用方法：
```PHP
$config = array( 
    'error' => FALSE, 
    'debug' => FALSE, 
    'log' => FALSE, ); 

 $this->load->model('sys/logic_dzz', 'oDzz'); 
 $this->oDzz->initialize($config); 

 $this->oDzz->topDepartment();  // 同步部门
 $this->oDzz->topAdmin();       // 同步用户
```
